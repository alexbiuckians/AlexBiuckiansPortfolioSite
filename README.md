About

A single-page site built from scratch with plain HTML, CSS, and vanilla JavaScript — no framework, no build step. It presents four flagship projects (GridCast, UpliftIQ, HouseEdge, EdgarIQ), a set of smaller projects, a skills overview, and contact details.

Featured projects


GridCast — hourly electricity-load forecasting shipped like production software (LightGBM, FastAPI, MLflow, Docker, CI). Cut MAE 72% over a seasonal-naive baseline.
UpliftIQ — causal uplift-modeling API that targets persuadable customers rather than just high-risk ones (causalml, econml, FastAPI, deployed on Render).
HouseEdge — adversarial game-integrity detection using sequential hypothesis testing (SPRT/CUSUM) and anomaly detection.
EdgarIQ — retrieval-augmented Q&A over SEC 10-K filings, running fully local and free (LangChain, ChromaDB, Ollama / Llama 3.2).
